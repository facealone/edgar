# Edgar Majordome

Edgar is a personal assistant which helps you to manage your house(s).

## Getting Started

### Server side

```bash
$ cd server
$ docker-compose up -d
```

API is available on https://api.edgar.local/

API doc is available on https://api.edgar.local/_doc

### Mobile side

```bash
$ cd mobile
$ npm install
$ npm start
```

## Technical stack

Edgar Majordome depends on the following technologies:

### Backend stack

-   Node.js
-   Nestjs
-   Typescript
-   Docker
-   PostgreSQL
-   Elasticsearch

### Mobile stack

-   ReactNative
-   Typescript
